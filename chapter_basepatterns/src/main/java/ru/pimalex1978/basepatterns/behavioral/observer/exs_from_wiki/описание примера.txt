Хотя классы библиотеки java.util.Observer и java.util.Observable существуют,
они объявлены устаревшими в Java 9, поскольку реализованная модель была весьма ограниченной.

Ниже приведен пример, написанный на Java, который принимает ввод с клавиатуры
и обрабатывает каждую строку ввода как событие. Когда строка предоставляется
из System.in, notifyObservers затем вызывается метод, чтобы уведомить всех
наблюдателей о возникновении события в форме вызова их методов «обновления».
