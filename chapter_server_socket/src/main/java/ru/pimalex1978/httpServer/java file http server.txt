Создание HTTP-сервера на Java
https://www.youtube.com/watch?v=0jmdSpykbIQ

https://github.com/EgorRepnikov/java-file-http-server

В данном примере показано, как создать HTTP-сервер
на чистой Java и на сокетах.
Итоговое приложение будет раздавать файлы из указанной
при запуске директории.
В создании http-сервера на Java используются только средства из
стандартной библиотеки (Java SE), к примеру, сокеты (socket).

Для проверки работы посылаемых щзапросов на сервер использовался Postman.
Результы сохранены в скиншотах в папке files/results.
Сервер работает.

MIME-тип (называемый "media type", а иногда "content type") - это строка,
отправляемая вместе с файлом, которая указывает тип файла. (например,
передаваемый аудиофайл может быть помечен как audio/ogg тип, а изображение
- image/png). MIME-тип играет точно такую же роль, как и расширение файла
в системе Windows. Когда HTTP-сообщение содержит Content-type заголовок,
тело запроса будет парситься в соответствии с MIME-типом, указанным
в заголовке.
